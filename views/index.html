CRM Email Validator

Validate CRM Emails with DNS & SMTP Checks

This module adds intelligent email validation to Odoo CRM leads. It helps reduce bounce rates and improves lead quality by validating email syntax, domain records, and SMTP server response.

✨ Key Features

  • Email Syntax Validation (RFC compliant)
  • DNS Record Validation (MX lookup)
  • SMTP Level Verification
  • Lead form indicators for valid/invalid emails
  • Smart 'Validate Email' button appears when DNS & syntax checks pass
  • Bulk Email Validation for CRM Leads
  • Filters: Valid, Invalid, and Not Yet Verified Emails
  • Fully configurable in Technical Settings
  • Supports library based DNS or external API (Coming Soon)

⚙ Configuration via Settings

Navigate to:

  1. Settings → Technical → Email → Outgoing Mail Servers

Fill The Details Like this:

Server Name: smtp

Priority: 1

From Filtering: abhi@gmail.com

Mail Server Settings

  • Authenticate with: Username
  • Connect to your server through your usual username
  • SSL Certificate: (Optional, based on provider)
  • Gmail OAuth Authentication (if applicable)
  • Connection Encryption: TLS (STARTTLS) or None
  • SMTP Server: bh-in-11.webhostbox.net
  • SMTP Port: 465

How It Works

  1. Email is validated using syntax rules
  2. DNS MX records are resolved
  3. SMTP handshake confirms mailbox existence
  4. Results are saved in the lead and filters are updated
